# Ergo - 产品需求文档 (PRD)

## 产品定位

Ergo（二狗）是一个基于 OpenClaw 的个人 AI 管家控制台，作为外部访问 OpenClaw 服务的统一入口，提供可视化的服务管理和状态监控能力。

## 目标用户

个人使用（terryzin），通过 cpolar 内网穿透从外部访问本地 OpenClaw 服务。

## 核心功能模块

### P0 - 已有 / 核心

| 模块 | 描述 | 当前状态 |
|------|------|----------|
| OpenClaw Dashboard 入口 | 跳转到 Gateway 管理界面 | 已完成 |
| Gateway 状态监控 | 实时检测 Gateway 运行状态 | 已完成（基础） |
| Cpolar 穿透信息 | 展示公网访问地址 | 已完成 |

### P1 - 计划开发

| 模块 | 描述 | 优先级 |
|------|------|--------|
| 项目文件管理 | 在线浏览、编辑工作空间文件 | 高 |
| 终端命令执行 | 远程执行系统命令 | 高 |
| 定时任务管理 | 查看/管理 Cron 任务及执行日志 | 中 |

### P2 - 远期规划

- Agent 对话入口（直接与 OpenClaw Agent 交互）
- 服务健康看板（聚合多个服务状态）
- 操作日志与审计

## 技术约束

- 前端: 纯静态页面，轻量无框架（当前），后续可引入前端框架
- 后端: 依赖 OpenClaw Gateway API（端口 18789）
- 部署: 本地运行 + cpolar 穿透对外
- 域名: `terryzin.cpolar.top`（Dashboard）/ `ergo-gateway.cpolar.top`（Gateway）

## 设计风格

Apple 风格深色主题，圆角卡片布局，毛玻璃效果头部。
