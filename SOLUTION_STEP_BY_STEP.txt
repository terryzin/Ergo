====================================

问题：https://terryzin.cpolar.top 返回 404

已确认：
✓ 本地服务 (8081) 正常运行
✓ cpolar 进程正在运行（3个进程）
✗ 隧道配置可能有问题

====================================
  解决方案 1：使用免费随机域名测试
====================================

如果 terryzin 子域名有问题，先用随机域名测试：

1. 打开命令提示符
2. 运行：
   cpolar http 8081 -region=cn

3. 看输出，找到类似这样的行：
   Forwarding    https://随机字符-cn.cpolar.top -> http://localhost:8081

4. 访问那个随机域名测试

如果随机域名能访问，说明：
- 本地服务正常 ✓
- cpolar 隧道正常 ✓
- 问题在于 terryzin 子域名配置 ✗

====================================
  解决方案 2：检查 terryzin 绑定
====================================

1. 访问: https://dashboard.cpolar.com/
2. 登录账号
3. 进入 "预留" → "二级域名"
4. 查看 terryzin 是否：
   - 已预留
   - 绑定到当前设备/Token

5. 如果未绑定，点击"绑定"

====================================
  解决方案 3：停止所有 cpolar 重新启动
====================================

1. 打开任务管理器 (Ctrl+Shift+Esc)
2. 找到所有 cpolar.exe 进程
3. 全部结束进程
4. 打开命令提示符
5. 运行：
   cd D:\.openclaw\workspace\my-dashboard
   cpolar http 8081 -subdomain=terryzin -region=cn

6. 查看输出是否显示：
   Forwarding    https://terryzin.cpolar.top -> http://localhost:8081
